# smart-home-ai-system
Smart Home AI system for sensor data simulation and analysis

This project simulates a smart home environment using Python.

## Features
- Simulated sensor data (temperature, humidity, energy usage)
- Basic AI-style rule system for warnings
- Simple modular structure (data + main)

## How it works
- data.py generates random sensor values
- main.py reads data and analyzes it
- system prints status (normal / warning)
